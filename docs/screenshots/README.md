# Screenshots

This directory holds screenshots referenced by the how-to guides under
[`docs/howto/`](../howto/). Most slots are currently empty — see
[`docs/howto/contributing-screenshots.md`](../howto/contributing-screenshots.md)
for the capture conventions, the list of filenames each how-to page
expects, and the scrubbing checklist contributors should follow before
committing an image.

## Reserved filenames

| File | Captured from | How-to page |
|---|---|---|
| `getting-started-main-window.png` | Main window | [getting-started.md](../howto/getting-started.md) |
| `settings-git-sync.png` | Settings → Git sync | [git-sync.md](../howto/git-sync.md) |
| `import-mremoteng-dialog.png` | Import dialog | [importing-from-mremoteng.md](../howto/importing-from-mremoteng.md) |
| `credentials-unlock.png` | Credentials dialog (locked) | [credentials.md](../howto/credentials.md) |
| `protocol-ssh-session.png` | Active SSH session | [protocols.md](../howto/protocols.md) |
| `plugins-dialog.png` | Plugins dialog | [plugins.md](../howto/plugins.md) |
| `backup-restore.png` | Backup or restore dialog | [backup-and-restore.md](../howto/backup-and-restore.md) |
| `update-check.png` | Update check result | [updates.md](../howto/updates.md) |
| `logs-viewer.png` | Log viewer dialog | [logs-and-diagnostics.md](../howto/logs-and-diagnostics.md) |
| `toolbar.png` | Main toolbar with tooltip visible | [getting-started.md](../howto/getting-started.md) |

If you add a new how-to page that needs a screenshot, also add the
filename here and in
[`contributing-screenshots.md`](../howto/contributing-screenshots.md).
